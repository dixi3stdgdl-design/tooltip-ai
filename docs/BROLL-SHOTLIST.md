# Lista de Tomas B-Roll para Demo de Tooltip AI

## Definición de B-Roll
B-Roll son las tomas complementarias que se usan para:
- Cubrir cortes en el audio principal
- Mostrar detalles importantes
- Añadir variedad visual
- Reforzar el mensaje
- Crear transiciones suaves

## CONSEJOS DE GRABACIÓN

### Calidad de Video
1. **Resolución**: Mínimo 1920x1080 (Full HD)
2. **Framerate**: 30 FPS para movimiento suave
3. **Formato**: MP4 (H.264) para compatibilidad
4. **Bitrate**: 8,000-10,000 kbps para calidad alta
5. **Color**: Espacio de color Rec. 709

### Movimiento de Cámara
1. **Estabilidad**: Usar trípode o estabilizador
2. **Velocidad**: Movimientos lentos y deliberados
3. **Dirección**: Siempre en misma dirección que lectura
4. **Enfoque**: Mantener enfoque en elemento clave
5. **Composición**: Regla de tercios para tomas amplias

### Iluminación
1. **Natural**: Ventana como luz principal
2. **Artificial**: Ring light para retratos
3. **Consistencia**: Mismas condiciones en todas las tomas
4. **Evitar**: Reflejos en pantalla, sombras duras
5. **Temperatura**: Luz blanca neutra (5000K)

### Audio
1. **Ambiente**: Silencio absoluto para tomas limpias
2. **Referencia**: Audio de referencia para sincronización
3. **Marcas**: Marcas de audio para edición
4. **Respaldo**: Grabar audio separado si es posible
5. **Monitoreo**: Usar auriculares para verificar calidad
